## Що це взагалі таке?

- **Репозиторій**: Збірка вправ і практичних робіт з курсу програмування (другий курс).
- **Мета**: Колекція лабораторних робіт та практичних завдань, організованих по папках.

**Швидкий старт**
- **Python**: Рекомендовано використовувати Python 3.10 або новіше.
- **Створити віртуальне середовище**: приклад для fish shell:

```fish
python -m venv .venv
source .venv/bin/activate.fish
python -m pip install --upgrade pip
```

- **Запуск проєкту**: запустіть `main.py` у потрібній папці. Приклади:

```fish
python CSLabaFirst/main.py
python CSLabaSecond/main.py
python CSLabaThird/main.py
python "CSLabaFourth-Fifth"/main.py
python TheFirstPractical/main.py
python TheSecondPractical/main.py
```

**Структура репозиторію**
- **`CSLabaFirst/`**: Перша лабораторна — містить `main.py` та супровідні файли.
- **`CSLabaSecond/`**: Друга лабораторна — містить `main.py` та матеріали.
- **`CSLabaThird/`**: Третя лабораторна — містить `main.py` та матеріали.
- **`CSLabaFourth-Fifth/`**: Четверта і п'ята лабораторні разом — містить `main.py` та супровідні файли.
- **`TheFirstPractical/`**: Практична №1 — містить `main.py` та супровідні файли.
- **`TheSecondPractical/`**: Практична №2 — містить `main.py` і підпапку `tasks/` з окремими файлами завдань.
- **`TheSecondPractical/tasks/`**: Файли завдань (наприклад, `TheFirst.py`, `TheSecond.py` тощо).
